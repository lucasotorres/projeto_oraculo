🔮 Finalizei mais um projeto acadêmico — e esse teve um toque especial.

Desenvolvemos uma aplicação web completa para o evento de portas abertas da FATEC Zona Leste, com o objetivo de receber alunos do ensino médio de forma criativa e funcional.

O projeto tem dois focos principais:

✅ Oráculo temático — Uma experiência interativa com mensagens aleatórias de vida pessoal, trabalho e estudos, com lógica para evitar repetição das últimas 3 mensagens exibidas.

✅ Captação de candidatos — Cadastro automático de visitantes com dados como nome, e-mail, telefone, bairro e curso de interesse, armazenando data e hora automaticamente.

Do ponto de vista técnico, o projeto envolveu:

🔹 Java 17 + Spring Boot, Spring Web e Spring Data JPA
🔹 Arquitetura MVC com princípios SOLID aplicados e comentados no código
🔹 MySQL com Triggers, Stored Procedures e UDFs para regras de negócio
🔹 Consultas com findBy, JPQL e Native Queries
🔹 Carga automática de dados via arquivos .txt no startup da aplicação
🔹 Painel administrativo com autenticação via Procedure (sem Spring Security)
🔹 Interface em JSP com Bootstrap

Foi um exercício completo de desenvolvimento — desde a modelagem (Diagrama de Classes e ER) até a entrega no GitHub com mvn site gerado.

Agradeço ao Prof. Leandro Colevati pela proposta desafiadora e bem estruturada! 🚀
